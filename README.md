
# ColdFusion Caveats

A field guide for CFML developers _by CFML developers_.

This project is intended to be a community-driven anthology of hard-won lessons from the ColdFusion developers who've luxuriated in both the pains and the pleasures of CFML.

This book isn't intended to compete with more educational resources, such as:

* Adobe: [Developing ColdFusion Applications](https://helpx.adobe.com/coldfusion/developing-applications/user-guide.html)
* Ortus Solutions: [Modern ColdFusion in 100 Minutes](https://www.ortussolutions.com/learn/books/coldfusion-in-100-minutes)
* Community: [Learn CF in a Week](https://www.learncfinaweek.com/)

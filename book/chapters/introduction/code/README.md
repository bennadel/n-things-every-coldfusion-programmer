
# Your ColdFusion Sandbox

This `code` folder is a chapter-specific sandbox in which you can try out your code in order to ensure that the CFML logic in the chapter is semantically and intentionally correct. This code is being run inside a Docker container - see the [`README.md`](../../../../README.md) file in the `book` root.

You can do whatever you want with `./app/` - this is the webroot that is loaded into the Docker container.
